Overview

WaveStrap is a lightweight Roblox launcher and runtime environment manager designed to control how Roblox is installed, prepared, and launched.

Instead of launching Roblox through the default browser bootstrap system, WaveStrap locates the Roblox client directly and launches it within a controlled environment. This allows WaveStrap to manage how Roblox starts, which version runs, and how the runtime environment is prepared.

Support for the Wave Executor

WaveStrap is designed to support the Wave executor by ensuring that Roblox launches in a predictable and stable environment.

Because Roblox updates frequently, launch behavior can change and disrupt tools that rely on consistent process startup. WaveStrap avoids this by starting the Roblox client directly instead of using the web bootstrap launcher.

ProcessStartInfo psi = new ProcessStartInfo { FileName = "RobloxPlayerBeta.exe", Arguments = "--app", UseShellExecute = true }; Process.Start(psi);

Version Management

WaveStrap can launch both the latest Roblox client and previously stored client builds. This helps maintain compatibility when new Roblox updates introduce breaking changes.

Downgraded clients are stored in a dedicated WaveStrap directory and can be launched directly whenever a specific version is needed for testing or compatibility.

%LOCALAPPDATA%\WaveStrap\WaveStrapDowngrade

Multi-Instance Support

Roblox normally restricts execution to a single running instance. WaveStrap prepares the runtime environment so multiple Roblox clients can run simultaneously when needed.

This is done by reserving synchronization objects that Roblox normally uses for single-instance enforcement before the client starts.

Mutex singletonEvent = new Mutex(true, "ROBLOX_singletonEvent"); Mutex singletonMutex = new Mutex(true, "ROBLOX_singletonMutex");

Installation Detection

WaveStrap searches multiple common Roblox installation paths to locate the client and ensure the correct executable is launched.

If Roblox is installed outside the preferred layout, WaveStrap can restore a more consistent runtime structure so launch behavior remains reliable.

%LOCALAPPDATA%\Roblox\Versions %LOCALAPPDATA%\Programs\Roblox Program Files\Roblox Program Files (x86)\Roblox

FixWave Repair Utility

WaveStrap includes a repair utility called FixWave that helps resolve runtime issues affecting Roblox or the Wave environment.

This can include repairing corrupted installations, restoring missing dependencies, and fixing environment issues that prevent Roblox or related tooling from working correctly.
