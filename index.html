WaveStrap
Roblox launch manager for the Wave ecosystem

Overview

WaveStrap is a lightweight Roblox launcher and runtime environment manager designed to control how Roblox is installed, prepared, and launched.

Instead of launching Roblox through the default browser bootstrap system, WaveStrap locates the Roblox client directly and launches it within a controlled environment. This allows WaveStrap to manage how Roblox starts, which version runs, and how the runtime environment is prepared.

Support for the Wave Executor

WaveStrap is designed to support the Wave executor by ensuring that Roblox launches in a predictable and stable environment.

Because Roblox updates frequently, launch behavior can change and disrupt tools that rely on consistent process startup. WaveStrap avoids this by starting the Roblox client directly instead of using the web bootstrap launcher.

ProcessStartInfo psi = new ProcessStartInfo
{
  FileName = "RobloxPlayerBeta.exe",
  Arguments = "--app",
  UseShellExecute = true
};

Process.Start(psi);

Version Management

WaveStrap can launch both the latest Roblox client and previously stored client builds. This allows compatibility to be maintained when new Roblox updates introduce breaking changes.

Downgraded clients are stored in the WaveStrap downgrade directory and can be launched directly when needed.

%LOCALAPPDATA%\\WaveStrap\\WaveStrapDowngrade

Multi-Instance Support

Roblox normally restricts execution to a single running instance. WaveStrap prepares the runtime environment so multiple Roblox clients can run simultaneously.

This is done by reserving synchronization objects used by Roblox for single-instance enforcement.

Mutex singletonEvent = new Mutex(true, "ROBLOX_singletonEvent");
Mutex singletonMutex = new Mutex(true, "ROBLOX_singletonMutex");

Installation Detection

WaveStrap automatically searches common Roblox installation locations to locate the client. If Roblox is installed outside the expected directory structure, WaveStrap can relocate the client to ensure a consistent runtime layout.

%LOCALAPPDATA%\\Roblox\\Versions
%LOCALAPPDATA%\\Programs\\Roblox
Program Files\\Roblox
Program Files (x86)\\Roblox

FixWave Repair Utility

WaveStrap includes a repair tool called FixWave which resolves runtime issues affecting Roblox or the Wave environment.

The repair utility can reset dependencies, reinstall required runtime components, and repair broken Roblox installations to ensure WaveStrap and the Wave executor function correctly.
